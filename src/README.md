# hello-devcontainer

Willkommen im hello-devcontainer-Projekt! Hier finden Sie eine Übersicht der Projektdateien:

- `.devcontainer`: Enthält die Konfiguration für den Entwicklungsumgebung.
    - `devcontainer.json`: Konfigurationsdatei für den DevContainer.
- `.idea`: IntelliJ IDEA spezifische Dateien.
- `out`: Ausgabeverzeichnis.
- `src`: Quellcodeverzeichnis.
    - `Main.java`: Hauptklasse des Projekts.
    - `README.md`: Dieses Markdown-Dokument.
- `.gitignore`: Dateien, die nicht in Git verfolgt werden sollten.
